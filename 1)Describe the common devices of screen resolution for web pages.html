Describe the common devices of screen resolution for web pages.

Answer:

Each different devices has a different screen resolution of web pages. Screen resolution refers to the clarity of the text and image displayed on the screen that the image on device screen is built up from thousands or millions of pixels. Nowadays, mobile phone and computer are common devices of screen resolution for web pages but this two devices are very different in screen resolution so some higher screen resolution web pages cannot be fully displayed through mobile phone because mobile phone with less screen resolution. Conclusion important the most popular screen sizes and resolutions when designing web page and mobile sites can make the user be more enjoyable.
