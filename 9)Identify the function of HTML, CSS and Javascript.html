Identify the function of HTML, CSS and Javascript

Answer:

HTML is the Hyper Text Markup Language for creating web pages and web applications, with Cascading Style Sheets and JavaScript in forms of technologies for the World Wide Web then web browsers receive HTML documents and render them into web pages. These websites can then be viewed by anyone else connected to the Internet. These Websites can then be view by anyone when connected to the internet.

CSS is the Cascading Style Sheets used for the presentation of a document written in a markup language. Most often used to set the visual style of web pages written in HTML. CSS designed primarily are beautifying the content, such as content layout, colour, fonts.

JavaScript is a programing language. It is used to make webpages interactive and provide online programs, including video games. JavaScript supports various function based implementations of patterns let the function may have properties and methods.
