Explain How CSS Colours representation in Hexadecimal

Answer:

The hex triplet is formed by concatenating three bytes in hexadecimal notation,such as red value , blue value , green value. triplet is a six-digit in hexadecimal format, three-byte hexadecimal number used in HTML, CSS, SVG, and other computing applications to represent colors on computer display. The hex triplet is obtained by concatenating the 6 hexadecimal digits together.
